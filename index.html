Git Utils

In the github pull request, you may not be interested on whole content of pullrequest because most of them are boilerplate code. In this extension, you can define rules !
You have two options:

Add pattern of which type of file you are going to exclude from pullrequest(no need to include '*').
Add pattern of which type of file you are going to include in pullrequest. you are able to choose only one option at a time.
