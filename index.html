GitHub Utils

In the github pull request, you may not be interested on whole content of pull request because most of them are boilerplate code. In this extension, you can define rules !
You have two options:

  1. Add pattern of which type of file you are going to exclude from pull request(no need to include '*').
  2. Add pattern of which type of file you are going to include in pull request.

Note: you are able to choose only one option at a time.
